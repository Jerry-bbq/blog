## 环境 

```bash
# 安装
npm install -g typescript
# 初始化
tsc --init
# 编译
tsc index.ts
# 编译并监视文件
tsc -w index.ts
# 编译并监视文件，同时生成声明文件
tsc -w --declaration index.ts
```

或者使用rollup
